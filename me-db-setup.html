SI-Einheiten Konzept wichtig

Originalwerte bleiben immer erhalten. Zusätzlich gibt es umgerechnete SI-Werte:

SpalteWoBedeutung
measurements.unitmeasurementsOriginale Einheit dieser Messung (z.B. mg/l bei russischem Labor)
measurements.value_simeasurementsWert umgerechnet in internationale Einheit
parameters.unit_siparametersInternationale Zieleinheit (z.B. mmol/l für Magnesium)
parameters.ref_range_low/high_siparametersReferenzbereich in SI-Einheit
Beispiel Magnesium: Russland misst in mg/l (20.98 mg/l = 0.863 mmol/l), Deutschland in mmol/l (0.93 mmol/l). Die App zeigt immer mmol/l im Dashboard und Trend.

Row Level Security (RLS)

lab_reports, staged_measurements, documents, chat_* und der Storage Bucket haben RLS-Policies (anon + authenticated). parameters und measurements haben RLS deaktiviert — der Anon-Key darf hier direkt lesen/schreiben/löschen (Einzelnutzer-Setup), daher funktioniert das volle Bearbeiten/Löschen im Frontend.

→

User schreibt Frage
→
Cloudflare Worker (health-db-ai)
Worker fügt hinzu:
→
System-Prompt + Kontext + ähnliche frühere Nachrichten
Worker ruft auf:
→
Claude API (claude-sonnet-4-5)
Antwort zurück
+
Speichern in chat_messages (ctx.waitUntil)
Voyage AI voyage-3
→
Embedding (1024 Zahlen) für Frage + Antwort

Gedächtnis-System (pgvector)

KomponenteDetail
Embedding-ModellVoyage AI voyage-3, 1024 Dimensionen
SpeicherortSupabase chat_messages.embedding (vector Typ)
SucheCosine Similarity — bei jeder neuen Frage werden die 5 ähnlichsten früheren Nachrichten geladen
Stand52 Nachrichten gesamt, 44 mit Embedding
Session-TitelEditierbar per Doppelklick, gespeichert in chat_sessions Tabelle
