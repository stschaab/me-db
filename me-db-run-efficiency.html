HR = f(Pace) · Steady-State Runs

Each point is a run (LDL / MDL / GA1 / SSL / Long Slow Distance) — no interval or sprint training. At a given fitness level, the points should fall on a single curve. If the curve shifts down over time (same pace, lower heart rate) or to the left (same heart rate, faster pace), that's a sign of improved aerobic fitness.
