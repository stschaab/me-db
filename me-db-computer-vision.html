🎥 Computer Vision

Automatische Zählerablesung per ONVIF-Kamera + Claude Vision (Haiku) · Stündlich, vollautomatisch, kein Cloud-Zugriff auf dein LAN nötig

⚡ So funktioniert es
  1. ONVIF-Kamera richtet sich auf deinen Zähler (Wasser, Strom, Gas)
  2. Python-Agent läuft auf deinem PC im Heimnetz, fragt stündlich die Kamera ab (lokaler HTTP-Snapshot)
  3. Foto wird als Base64 an Claude Haiku (Vision) geschickt → liest den Zählerstand ab
  4. Wert wird in Supabase gespeichert → hier im Dashboard sichtbar & im Trend Viewer
💡 Der PC muss durchlaufen (Bildschirmsperre reicht, kein Shutdown) · Anthropic API Key nötig: console.anthropic.com → API Keys · Kosten: ~0,001 € / Ablesung mit Claude Haiku
🔧 Agenten konfigurieren & Skript herunterladen

Formular ausfüllen → PowerShell-Skript herunterladen → auf dem Ziel-PC als Administrator ausführen.
Das Skript installiert Python-Pakete, schreibt den Agenten und richtet den Windows Task Scheduler ein.

🗄️ Supabase Tabelle (bereits erstellt)

Die Tabelle meter_readings wurde bereits angelegt. Falls du sie manuell neu erstellen musst: 

CREATE TABLE IF NOT EXISTS meter_readings (
  id           uuid DEFAULT gen_random_uuid() PRIMARY KEY,
  agent_name   text NOT NULL,
  meter_type   text NOT NULL DEFAULT 'Wasserzähler',
  value        numeric NOT NULL,
  unit         text NOT NULL DEFAULT 'm3',
  raw_response text,
  captured_at  timestamptz NOT NULL DEFAULT now(),
  created_at   timestamptz NOT NULL DEFAULT now()
);
CREATE INDEX IF NOT EXISTS idx_meter_readings_agent_date
  ON meter_readings (agent_name, captured_at DESC);
ALTER TABLE meter_readings ENABLE ROW LEVEL SECURITY;
CREATE POLICY "anon_insert" ON meter_readings FOR INSERT TO anon        WITH CHECK (true);
CREATE POLICY "auth_all"    ON meter_readings FOR ALL    TO authenticated USING (true) WITH CHECK (true);
